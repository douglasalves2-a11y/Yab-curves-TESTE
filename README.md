# YAB CURVES MODA PLUS SIZE

Bem-vindo ao repositório oficial da **YAB CURVES MODA PLUS SIZE** 💜

A YAB Curves é uma loja online especializada em moda plus size feminina, oferecendo peças modernas,
 confortáveis e estilosas para valorizar a diversidade de corpos e promover autoestima através da moda.

## ✨ Sobre o projeto

Este site foi desenvolvido para proporcionar uma experiência de compra prática,
intuitiva e agradável para clientes que buscam:

- Roupas plus size modernas
- Looks casuais e elegantes
- Novidades e tendências
- Facilidade de navegação
- Compra segura online

## 🚀 Funcionalidades

- Página inicial com destaques e promoções
- Catálogo de produtos organizado por categorias
- Página individual de produtos
- Área de contato
- Integração com redes sociais
- Design responsivo para mobile, tablet e desktop
